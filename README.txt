Overview
This is my personal portfolio website, where I present my skills, projects, and experience. It serves as a central hub for potential employers, clients, and collaborators to learn more about me and my work.

Features
About Me: A brief introduction and background.
Projects: A showcase of my projects with descriptions and links to the code or live demos.
Skills: A list of technical skills and competencies.
Contact: Contact information and a contact form for inquiries.

Technologies Used
Frontend: HTML, CSS, JavaScript
Tools: Git, GitHub, Visual Studio Code

Installation
To run the website locally, follow these steps:

	1. Clone the repository: git clone https://github.com/Truong-NguyenData/MyWebsite.git

	2. Navigate to the project directory: cd MyWebsite

	3. Open index.html in your preferred web browser.

Usage
	Online: You can visit the live version of the website at .
	Locally: Open index.html in your web browser after following the installation steps.
